      Copy the agents to your Claude Code directory to make them available in all your sessions.
      
      ```bash
      cp -r agency-agents/* ~/.claude/agents/
      ```
    </Step>
    
    <Step title="Activate an agent">
      Reference any agent in your Claude Code sessions by name.
      
      ```
      Hey Claude, activate Frontend Developer mode and help me build a React component
      ```
    </Step>
    
    <Step title="Use NEXUS for multi-agent workflows">
      For complex projects, activate the Agents Orchestrator to coordinate multiple specialists.
      
      ```
      Activate Agents Orchestrator in NEXUS-Sprint mode to build [your feature]
      ```
      
      <Accordion title="NEXUS deployment modes">
        - **NEXUS-Full**: Complete product lifecycle (12-24 weeks, all agents)
        - **NEXUS-Sprint**: Feature or MVP build (2-6 weeks, 15-25 agents)
        - **NEXUS-Micro**: Targeted task execution (1-5 days, 5-10 agents)
      </Accordion>
    </Step>
  </Steps>
